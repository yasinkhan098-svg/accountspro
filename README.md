# LedgerX - Advanced Accounting Web App

A powerful, keyboard-driven accounting web application inspired by Tally Prime.

## Features
- Multi-Company management
- Tally-style keyboard navigation (Enter-as-Tab, Shortcuts)
- Full Accounting Masters (Ledgers, Groups, Currencies)
- Inventory Management (Stock Items, Groups, Units, Godowns)
- Voucher Entry (Sales, Purchase, Contra, Payment, Receipt, Journal)
- Real-time Reports (Balance Sheet, Profit & Loss, Trial Balance, Day Book)
- GST Support & Invoice Printing
- Security Control (Company-level Passwords)

## Tech Stack
- Next.js 14
- Prisma ORM
- PostgreSQL
- Custom Auth System

## Deployment
This project is ready to be deployed on Vercel with a PostgreSQL database (Neon/Supabase).
